# 소송서면 증거번호 자동 정렬기

서면(`.hwpx`) 파일을 수정하다 보면 **증거**의 번호 순서가 바뀌는 경우나, 증거가 삽입되고 삭제되기도 합니다.
이 도구는 본문에 등장하는 순서대로 자동으로 번호를 재정렬해주고 서면 마무리 부분(입증방법, 소명방법, 첨부서류 등)에 정리합니다.
또한 새로운 증거가 삽입되었는데 서면 마무리 부분에만 있고 본문에 없을 경우 해당 내용을 경고하는 파이썬 스크립트입니다.

---

## 어떤 일을 해주나요?

- 본문에서 **증거**가 처음 등장하는 순서대로 1번부터 다시 번호를 매김
- 같은 증거가 본문에 여러 번 나오면 → 번호는 그대로 유지 (고정)
- 문서 끝의 **증거목록**(ex: 입증방법, 소명방법, 참고자료 등)도 새 순서대로 자동 재생성
- 본문에 없는 증거가 목록에만 있으면 **경고 메시지**로 알려줌
- **소장**(갑 제N호증), **준비서면**(을 제N호증), **변호인의견서**(참고자료 N) 세 가지 형식 모두 지원

### 예시

소장은 본문에 `갑 제N호증`을, 피고 준비서면은 `을 제N호증`을 인용하고, 변호인의견서는 본문 아래에 `- 참고자료 N.`을 첨부합니다.
세 형식 모두 본문 등장 순서대로 번호를 1번부터 다시 매기고, 마무리 목록도 자동 재생성합니다.

## 사용 전 준비

### 1. Python 설치
- https://www.python.org/downloads/ 접속
- **Download Python 3.x.x** 클릭 → 설치
- 설치 화면에서 **"Add Python to PATH"** 반드시 체크 후 Install

### 2. 코드 받기
이 페이지 상단의 초록색 **`<> Code`** 버튼 → **Download ZIP** → 압축 풀기

(Git을 설치했다면: `git clone https://github.com/사용자명/저장소명`)

## 사용 방법

### 단계 1. 소장 파일을 코드 폴더에 넣기

다운받은 폴더 안에 정렬하고 싶은 `.hwpx` 소장 파일을 복사해 넣어주세요.

### 단계 2. PowerShell 열기

폴더 안에서 **빈 공간 우클릭 → "터미널에서 열기"** (Windows 11)
또는 Shift + 우클릭 → "PowerShell 창 여기서 열기" (Windows 10)

### 단계 3. 미리보기 (실제 파일은 안 바뀜)

**소장 (갑 제N호증):**
```powershell
$env:PYTHONIOENCODING="utf-8"
python exhibit_renumber.py --preview "소장파일.hwpx"
```

**준비서면 (을 제N호증):**
```powershell
$env:PYTHONIOENCODING="utf-8"
python exhibit_renumber_defendant.py --preview "준비서면파일.hwpx"
```

**변호인의견서 (참고자료 N):**
```powershell
$env:PYTHONIOENCODING="utf-8"
python reference_renumber.py --preview "의견서파일.hwpx"
```

번호가 어떻게 바뀔지 미리 확인할 수 있어요.

### 단계 4. 실제 변환

**소장:**
```powershell
python exhibit_renumber.py "소장파일.hwpx"
```

**준비서면:**
```powershell
python exhibit_renumber_defendant.py "준비서면파일.hwpx"
```

**변호인의견서:**
```powershell
python reference_renumber.py "의견서파일.hwpx"
```

같은 폴더에 **`원본파일명_숫자정렬완.hwpx`** 가 새로 생성됩니다. 원본은 그대로 유지돼요.

## 자주 묻는 질문

**Q. 한글이 깨져 보여요**
A. PowerShell에서 `$env:PYTHONIOENCODING="utf-8"` 명령을 먼저 입력하세요.

**Q. "찾을 수 없습니다" 에러가 나요**
A. 폴더 경로에 대괄호(`[]`)가 있으면 `cd` 대신 `Set-Location -LiteralPath '경로'`를 써주세요.

**Q. 입증방법 목록의 1번, 2번이 안 보여요**
A. 한글 파일에서 자동번호 매기기 스타일이라면 텍스트에는 번호가 없어도 화면에서는 자동으로 보입니다. 정상입니다.

**Q. `.hwp` 파일도 되나요?**
A. 아니요. `.hwpx` (한글 2010 이후 형식)만 지원합니다.

## 작동 원리

1. HWPX(ZIP) 안의 `Contents/section0.xml` 파싱
2. **증거목록**(ex: 입증방법, 소명방법, 첨부서류 등) 섹션 자동 탐지 (오탐 방지를 위해 단락 단독 키워드 + 다음 줄 패턴 두 조건 확인)
3. 입증방법 **이전** 본문에서 증거 최초 등장 순서 수집
4. 새 번호 매핑 생성 후 본문 전체 치환
5. 입증방법 목록을 한글 자동번호 스타일로 통일 재생성
6. 새 HWPX 파일로 저장
